# Description 
Analog to Digital conversion using DMA (Direct Memory Access) the data read by analog pin is transferred to PC using UART communication through buffer.

# PIN Connection 
|Pin | Function |
| ------ | ------ |
|PA2 | UART2_Tx|
|PA3 | UART2_Rx|
|PA4 | Analog Sense Pin|
|PC13 | Status LED|
